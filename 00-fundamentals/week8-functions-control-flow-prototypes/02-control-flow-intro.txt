Estructuras de Control de Flujo - Control Flow (If, for, while, etc)
  son un mecanismo para cambiar el comportamiento de ejecucion de los scripts
    por defecto se ejecuta de arriba hacia abajo y linea por linea

estructuras básicas de control de flujo en JavaScript
  condicionales
    if, else if, else

  bucles/loops
    for
      ejecuta un bloque de código un número específico de veces

    while
      ejecuta un bloque de código mientras una condición especificada sea verdadera

    do...while
      similar a while, pero ejecuta el bloque de código al menos una vez antes de verificar la condición

    for...of
      itera sobre los valores de un objeto iterable

    for...in
      itera sobre las propiedades enumerables de un objeto

  seleccion multiple / switch case
    'switch' evalua una expresión y luego
    se ejecuta el código asociado a una etiqueta 'case' en caso los valores coincidan
