bucles
  son una herramienta de control de flujo
  que repetirán un bloque de código mientras se cumpla una condición específica

  en js los mas conocidos son
    for, while y do...while

bucle for
  for (let i = 0; i < 5; i++) {
    console.log(i); // Imprimirá los números del 0 al 4
  }
bucle while
  let i = 0;
  while (i < 5) {
    console.log(i); // Imprimirá los números del 0 al 4
    i++;
  }
bucle do...while
  similar al while pero con una diferencia
    el bloque de código se ejecuta al menos una vez antes de que se compruebe la condición

  let i = 0;
  do {
    console.log(i); // Imprimirá los números del 0 al 4
    i++;
  } while (i < 5);

break y continue
  break
    "rompe" el bucle y termina su ejecución inmediatamente
      desplaza el control del programa a la siguiente línea después del bucle

    usar cuando queires salir del bucle antes de que haya completado todas sus iteraciones

  continue
    "salta" una iteración en el bucle
      cuando el bucle se encuentra con continue
        omite el resto del código que se encuentra en esa iteración y pasa directamente a la siguiente

consideraciones finales
  usa un bucle for cuando
    sepas cuántas veces quieres que se repita el código

  usa while o do...while cuando
    la cantidad de repeticiones se determine por una condición
      que no está necesariamente ligada a un contador
