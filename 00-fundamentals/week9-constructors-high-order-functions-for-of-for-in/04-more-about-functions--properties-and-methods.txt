las funciones en JavaScript son objetos
    significa que pueden tener propiedades y métodos

cuando creamos una función
  el objeto que se crea tiene propiedades predefinidas

Propiedades de funciones

  1. name
      propiedad apunta al nombre de una función
        function holaMundo() {}
        console.log(holaMundo.name); // "holaMundo"

  2. length
      retorna con cuántos parámetros requeridos se declaró la función
        function sumar(a, b, c = 0) {
          return a + b + c;
        }

        console.log(sumar.length); // 2 -- el parámetro 'c' tiene un valor por defecto asi que tampoco cuenta

  3. arguments
      este objeto contiene todos los argumentos que se pasaron a una función
        aunque es útil, su uso ha sido desplazado en favor de los parámetros "rest"

        no toma en cuenta la cantidad de parámetros declarados en la función
          sino la cantidad de argumentos enviados en la invocación de la misma

        este objeto no esta disponible en funciones de flecha

        function mostrarArgumentos() {
          console.log(arguments) // <- imprimimos el valor del objeto arguments
        }

        mostrarArgumentos(1, 'hola', true, { escuela: "Codeable" });

        /* Imprime:
        {
          '0': 1,
          '1': 'hola',
          '2': true,
          '3': { escuela: 'Codeable' },
          length: 4
        }
        */

Métodos del Prototipo de Función

  estos metodos existen gracias al Prototipo de Función (Function Prototype)

  estos son solo los metodos mas relevantes

  1. bind()
      permite establecer el valor de this de una función y crear una nueva función con ese contexto

      ejemplo
        function saludar() {
          console.log(`¡Hola! Mi nombre es ${this.nombre}.`);
        }

        const saludo = saludar.bind({nombre: "Juan"});

        saludo(); // ¡Hola! Mi nombre es Juan.

  2. call()
      permite llamar a una función con un objeto específico como this y con argumentos individuales
        no se crea una nueva función
          sino que reusa una función existente
            pasando el objeto al que apuntará this y los argumentos necesarios

      ejemplo
        function presentar(titulo, ciudad) {
          console.log(`${titulo} ${this.nombre}, de ${ciudad}.`);
        }

        const persona = { nombre: "Juan" };
        presentar.call(persona, "Sr.", "Lima"); // Sr. Juan, de Lima.

  3. apply()
      similar a call()
        la diferencia es que los argumentos los pasamos como un array
          en lugar de argumentos individuales

      ejemplo
        presentar.apply(persona, ["Sr.", "Lima"]); // Sr. Juan, de Lima.
