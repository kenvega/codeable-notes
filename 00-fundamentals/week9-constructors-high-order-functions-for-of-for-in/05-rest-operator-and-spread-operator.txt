Rest Parameters y Spread Syntax
  introducidas con ES6 (en 2015)
  facilitan el trabajo con múltiples argumentos, arreglos y objetos

rest parameters (tambien llamados 'resto de parametros')
  se usa en funciones

  permite capturar un numero indefinido de argumentos y agruparlos en un array

  ejemplo de uso
    si quisieramos una funcion que nos permita sumar todos los argumentos
      independientemente de cuantos argumentos se le pase a una funcion

        function sumarTodos(...numeros) {
          let result = 0;

          for(let i = 0; i < numeros.length; i++) {
            result += numeros[i]
          }

          return result;
        }

        console.log(sumarTodos(1, 2, 3)); // 6
        console.log(sumarTodos(10, 20, 30, 40, 50)); // 150

    tambien se puede combinar paraemtros requeridos
      parámetros opcionales (con valor por defecto) y el "resto" de parámetros (con la sintaxis ...)

        function ejemplo(a, b, c = 0, ...resto) {
          console.log("Requeridos: ", a, b);
          console.log("Opcional: ", c);
          console.log("Resto: ", resto);
        }

        ejemplo(10, 20, 30, 40, 50);
        /* Imprime
        Requeridos: 10 20
        Opcional: 30
        Resto: [40, 50]
        */

      el nombre de la variable luego de los tres puntos (...) es irrelevante (puede ser cualquier nombre)
        es muy común utilizar ...rest o ...args

spread syntax (uso es muy común en arreglos y objetos)

  si el operador "rest" nos permitía agrupar elementos
  el operador "spread" nos permite desagrupar o expandir elementos agrupados

  uso en arreglos
    recuerda: un arreglo es un conjunto o lista de elementos ordenados
    el operador spread (...) nos permite desagrupar el arreglo en elementos individuales

      // Para crear un arreglo incluyendo los elementos de otro arreglo

      const arr1 = [1, 2, 3];
      const arr2 = [...arr1, 4, 5];
      console.log(arr2); // [1, 2, 3, 4, 5]

      // Para que los elementos de un arreglo pasen como argumentos individuales a una función

      let suma = (a, b) => a + b;

      const nums = [10, 30]
      console.log(suma(...nums)); // 40

  uso en objetos
    recuerda: un objeto es un conjunto de propiedades
    el operador spread (...) nos permite descomponer/expandir un objeto en sus propiedades individuales

      // Para crear un objeto a partir de las propiedades de otro

      const datos1 = { nombre: "Juan", apellido: "Perez" };
      const persona = { ...datos1, pais: "Mexico", ciudad: "Guadalajara" }

      console.log(persona); // {nombre: 'Juan', apellido: 'Perez', pais: 'Mexico', ciudad: 'Guadalajara'}
