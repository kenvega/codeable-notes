consultas SQL (SQL queries)
  SQL es el lenguaje para
    generar, manipular y recuperar datos de una base de datos relacional

que se aprendio en esta clase
  recuperar datos con la cláusula SELECT - FROM
  agregar restricciones con la cláusula WHERE y operadores SQL (=, !=, LIKE, NOT LIKE, %, _, IN, NOT IN)
  ordenar los resultados con ORDER BY
  filtrar datos usando DISTINCT
  limitar el número de filas a mostrar con LIMIT y OFFSET
  crear salidas personalizadas con la cláusula CASE
