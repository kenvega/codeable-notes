diferencias de jsx con html
  atributos
  se convierten en un objeto props que se pasa al React.createElement

  todas las reglas de atributos html que se usan en las propiedades de etiqueta
    se aplican en este caso

  se usa className en vez de class

  los atributos estan ahora en camelCase
    como className y como tabIndex en lugar de tab-index

  todos los elementos tienen que tener una etiqueta de cierre

    en html el elemento img se puede escribir sin etiqueta de cierre
      tambien pasa con el input
