2 casos de uso de useMemo

  - para recordar entre renders un valor dificil de calcular

  - para no re-renderizar si un prop realmente no cambia en valor
      recuerda que
        const a = { value: 'test' };
        const b = { value: 'test' };
        console.log(a === b); // false


      entonces si ese fuese un prop igual se haria un re-render
        incluso si usaramos react.memo se haria un re-render
          porque la comparacion daria false

            en ese caso se usa usememo

            para que la comparacion se haga como
              const a = { value: 'test' };
              const b = a;
