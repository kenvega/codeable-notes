vscode live server extension

  es mejor usar esta extension a solo abrir un archivo html directamente en el navegador
    porque cuando querramos traer css o imagenes sera mas sencillo

otras extensiones recomendadas
  prettier, live server, live share

elementos html

  links - anchors - <a />

    por el atributo rel puedes saber cual es la ventana hija y la ventana padre
      esto se usa para medios de pago con tarjeta por ejemplo

  imagenes

    el atributo alt es muy util para el SEO de la pagina
      los motores de busqueda no entienden imagenes
        pero con alt al menos se dan una idea de asociar que estas mostrando con lo que se busca un usuario

  pre y code

    pre
      para respetar saltos de linea

    code
      formatear texto como codigo
