como se centran elementos

  margin auto
    creo que eso solo centra elementos que tengan display block
      seria que pruebes si es que centra elementos que tengan display inline

  text-align: center
    para centrar el texto dentro de una caja

  centrar con flex
    display: flex;
    justify-content: center; // para centrar de manera horizontal
    align items: center; // para centrar de manera vertical

    se necesita una altura fija para el elemento que vayas a centrar con display flex
      para que puedas centrar

    estas propiedades se le aplican al contenedor, no al contenido que se esta centrando

    estan en box-model/lab/center-box-text
